# Olas Glassmontering

En enkel og profesjonell nettside for Olas Glassmontering bygget med Next.js og Tailwind CSS.

## Funksjoner

- Hero-seksjon med firmalogo og slagord
- Informasjonsseksjon om firmaet
- Referanseseksjon med eksempler på tidligere prosjekter
- Kontaktskjema
- Footer med kontaktinfo og organisasjonsnummer

## Komme i gang

1. Installer avhengigheter:
```bash
npm install
```

2. Kjør utviklingsserveren:
```bash
npm run dev
```

3. Åpne [http://localhost:3000](http://localhost:3000) i nettleseren.

## Bygging for produksjon

```bash
npm run build
npm start
```

## Teknologi

- Next.js 14 (App Router)
- React 18
- Tailwind CSS 3
- TypeScript
